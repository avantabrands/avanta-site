# Avanta Brands 30 / 60 / 90 Day SEO + Entity + LLM Plan

## Goal

Make `avantabrands.co` the clearest, most credible TikTok Shop growth agency result for:

- branded queries around `Avanta Brands`
- commercial TikTok Shop agency searches
- recommendation-style LLM prompts
- proof-seeking buyers comparing agencies

This plan is designed to stay **safe**, **measured**, and **free-first**.

## Core Operating Rules

- Do not mass publish thin pages.
- Do not fake reviews, testimonials, or case studies.
- Do not overuse indexing requests.
- Publish pages only when they are commercially useful, proof-backed, or strongly differentiated.
- Favor entity clarity, proof density, and premium presentation over volume.

## Free Tool Stack

Use these first before paying for anything:

- Google Search Console: indexing, performance, coverage
- Google Search Console API: reporting automation
- Google Analytics: user behavior and conversions
- Bing Webmaster Tools: extra crawl/indexing visibility
- Microsoft Clarity: free session recordings and heatmaps
- Google Business Profile: only if Avanta has a legitimate business presence to represent
- Cloudflare free tools: caching, redirects, basic bot visibility, analytics
- GitHub: publishing history and rollback safety

No paid SEO tool is required yet. We should only consider one later if the free stack stops answering important questions.

## Positioning Rules

Every important page should reinforce:

- `Avanta Brands`
- `TikTok Shop growth agency`
- `creator + affiliate + paid media + listing optimization`
- proof-backed execution for brands

The site should avoid vague brand language that could be confused with unrelated `Avanta` businesses.

## Indexing Strategy

### Submit once

- `https://avantabrands.co/sitemap.xml`

### Manually request indexing only for:

- homepage

- case study hub
- strongest case studies

### Do not do:

- repeated requests on unchanged pages
- indexing every URL every day

## LLM Optimization Rules

We cannot force LLMs to recommend Avanta, but we can improve the odds by making the site easier to cite and trust.

### Pages that matter most for LLMs

- homepage
- about
- process
- strongest case studies
- “what does a TikTok Shop agency do”
- “how to choose a TikTok Shop agency”
- vertical service pages

### What these pages need

- clear answers
- clear brand/entity language
- real proof
- visible specifics
- no vague or awkward SEO copy
- premium formatting on mobile and desktop

## KPI Targets

Track these weekly:

- indexed Tier 1 URLs
- branded query impressions
- commercial query impressions
- CTR on homepage and service pages
- case-study page clicks
- booked calls from organic or direct assisted paths

## When To Consider A Paid Tool

Only consider a paid tool if:

- Search Console + Clarity + Analytics stop answering key questions
- we need structured backlink prospecting at scale
- we need SERP monitoring that free tools cannot cover

If we ever add one, start with a short free trial only and use it to:

- export query gaps
- export competitor title/meta patterns
- export backlink opportunities

Then cancel unless it is clearly earning its keep.
